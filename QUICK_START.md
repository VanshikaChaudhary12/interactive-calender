# 🚀 Quick Start Guide - Enhanced Features

## Welcome to Your Enhanced Calendar! 🎉

Your calendar now includes powerful new features to make planning easier and more enjoyable.

---

## 🔍 Search Your Notes

**How to use:**
1. Look for the search bar at the top of the Notes panel
2. Type any word or phrase
3. Results filter instantly as you type
4. Click the ❌ to clear your search

**Pro tip:** Search works across all your notes, not just the selected date range!

---

## 🏷️ Organize with Tags

**Adding tags to notes:**
1. Select a date range
2. Write your note
3. Click on tag buttons (Work, Personal, Urgent, Event, Reminder)
4. Selected tags turn colored
5. Click "Add Note" - your tags are saved!

**Filtering by tags:**
1. Click "Filter Tags" button
2. Select one or more tags
3. Only notes with those tags appear
4. Click "Clear" to see all notes again

**Tag Colors:**
- 🔵 **Work** - Professional tasks
- 🟢 **Personal** - Personal activities
- 🔴 **Urgent** - High priority items
- 🟣 **Event** - Special occasions
- 🟡 **Reminder** - Things to remember

---

## ⚡ Quick Actions Menu

**The floating button in the bottom-right corner:**

Click it to reveal 4 powerful actions:

1. **↩️ Undo** - Made a mistake? Click to undo
2. **↪️ Redo** - Changed your mind? Click to redo
3. **📥 Export** - Download all notes as JSON file
4. **🖨️ Print** - Print your calendar with notes

**Pro tip:** Undo/Redo work for note additions, edits, and deletions!

## 📊 Statistics Dashboard

**At the top of your calendar, you'll see 4 cards:**

- **📝 Total Notes** - How many notes you've created
- **📅 This Month** - Notes in the current month
- **📈 Active Days** - Days with at least one note
- **⏱️ Avg/Day** - Average notes per active day

**Use it to:** Track your planning habits and stay motivated!

## 📥 Export Your Data

### Export as JSON:
1. Click the floating action button (bottom-right)
2. Click "Export"
3. JSON file downloads automatically
4. Contains all notes with dates and tags

### Print Calendar:
1. Click the floating action button
2. Click "Print"
3. New window opens with formatted calendar
4. Use browser's print function (Ctrl+P)
5. Save as PDF or print to paper

**Use cases:**
- Backup your data
- Share with others
- Archive old notes
- Print for offline use

## 🐛 Troubleshooting

### Search not working?
- Make sure you're typing in the search bar (not note input)
- Search is case-insensitive
- Try clearing filters

### Tags not showing?
- Tags only appear on notes created after this update
- Old notes won't have tags (this is normal)
- You can edit old notes to add tags

### Undo button disabled?
- No history to undo
- Undo only works for note changes
- Refresh clears undo history

### Export not downloading?
- Check browser's download settings
- Allow pop-ups for this site
- Try a different browser
